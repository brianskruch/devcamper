# devcamper  API
> Devcamper from Node.js Masterclass With Express &amp; MongoDB by Traversy Media
> 
## Usage

Rename "config/config_TEMPLATE.env" to "config/config.env" and update value settings to your own.

## Install Dependencies
```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
